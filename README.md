# Java Inventory Control System

A simple inventory control system developed in Java.

This project allows users to register products, list items, sell products, update stock quantity, remove products, and check low-stock items.

## Features

* Register products
* List all products
* Sell products
* Update stock quantity
* Remove products
* Show low-stock products

## Technologies Used

* Java
* ArrayList
* Scanner
* Object-Oriented Programming

## Project Purpose

This project was created to practice basic Java programming concepts, including classes, objects, methods, loops, conditionals, user input, and list manipulation.

## How to Run

1. Clone this repository.
2. Open the project in IntelliJ IDEA or another Java IDE.
3. Run the `Main.java` file.
4. Use the terminal menu to interact with the system.

## Example

The system allows the user to manage products with information such as code, name, price, and quantity.
